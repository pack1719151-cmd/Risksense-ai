# RiskSense AI

RiskSense AI is an automated, explainable risk intelligence engine
that detects anomalies and compliance risks in business data.

## Problem
Companies lose money due to silent data errors, anomalies,
and compliance risks that go unnoticed.

## Solution
RiskSense AI continuously monitors data, detects risks,
and explains them in plain English.

## Tech Stack
- Python
- Pandas / NumPy
- Ubuntu Linux
