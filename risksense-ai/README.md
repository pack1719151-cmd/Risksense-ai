# RiskSense AI

RiskSense AI is an enterprise-grade, explainable AI system that detects
business, data, and compliance risks before they cause financial loss.

## Key Capabilities
- AI-based anomaly detection
- Rule-based compliance checks
- Explainable risk reasoning
- Linux-native automation
- Report generation

## Tech Stack
- Python
- Pandas / NumPy
- Scikit-learn
- Ubuntu Linux
